#### Features
- Authentication
- server side and client side rendering
- single end point graph API implementation for efficient CRUD operations
- nextJS api routes

#### Frontned :
- React
- Apollo client
- tailwindcss

#### Backend :
- NextJS API routes
- Apollo server
- GraphQL
- prisma
- PostgreSQL

Install all dependencies
```bash
yarn install
```

run the application
```bash
yarn dev
```
